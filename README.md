# Hotel-Booking-Dashboard

This project presents an interactive Hotel Booking Dashboard built using Microsoft Excel to analyze hotel booking performance for the years 2022–2023.

The dashboard transforms raw booking data into clear visual insights to help track revenue, bookings, room availability, and customer behavior.

Business Objective

The goal of this dashboard is to:

Monitor overall hotel performance

Identify top-performing hotels and cities

Analyze booking trends over time

Compare room availability vs. room demand

Understand customer segmentation (New vs Existing)

Tools & Techniques Used

Pivot Tables

Pivot Charts

Excel Formulas

KPI Cards (Summary Metrics)

Slicers (Year filter for interactivity)

Data Sorting & Aggregation

 Key Performance Indicators (KPIs)

Total Profit: 11M

Total Bookings: 3370

Total Rooms: 15K

Hotel Count: 140

 Dashboard Analysis & Insights
 Top Performing Hotel

Marine Resort has the highest number of bookings (1004), followed by Hames Palace and View Hotel.

 Bookings by Country

India represents the highest booking percentage (26%), followed by Australia (22%).

 Customer Segmentation

68% Existing Customers

32% New Customers
This indicates strong customer retention.

 Monthly Profit Trend

Highest profit recorded in December

Noticeable fluctuations during mid-year months

Q4 shows stronger performance overall

 Room Type Performance

Sharing rooms have the highest booking volume compared to other room types like Deluxe, Business Class, and Luxury.

 Room Availability by City

Sydney and Mumbai show high availability levels, indicating large hotel capacity in these cities.

Presenting business insights clearly and visually

 Conclusion

This dashboard demonstrates how Excel can be used as a powerful Business Intelligence tool to analyze booking performance and support decision-making.
